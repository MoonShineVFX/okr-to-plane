# 快速開始：OKR to Plane Skill（表格格式版）

## 🎯 這個 Skill 解決什麼問題？

將你的表格格式 OKR Markdown 文件**一鍵匯入**到 Plane 專案管理系統，並使用 Cycles 進行季度追蹤。

**特別功能：** 跨季度 OKR 的**互動式拆分**！

## ⚡ 5 分鐘上手

### 步驟 1: 安裝 Skill

1. 下載 `okr-to-plane.zip`
2. 在 Claude.ai 中：Settings > Capabilities > Skills > Upload skill
3. 選擇 ZIP 檔案並啟用

### 步驟 2: 確認 Plane MCP 已連接

Settings > Extensions > 確認 Plane 顯示 "Connected"

### 步驟 3: 測試匯入

**複製以下 OKR 範例：**

```markdown
### Objective 1：建立 4DGS 能力 (Q1-Q2)

| KR  | 關鍵成果        | 衡量指標     |
| --- | ----------- | -------- |
| KR1 | 完成 Blender 培訓 | 產出 2 個模型  |
| KR2 | 建立標準工作流     | 產出 SOP 文件 |

### Objective A：AI 輔助開發 (Q1-Q4)

| KR  | 關鍵成果           |
| --- | -------------- |
| KR1 | 學習使用 Skills    |
| KR2 | 學習自架 MCP       |
| KR3 | 學習全自動開發流程      |
```

**在 Claude 中輸入：**

```
請幫我把這份 OKR 匯入到 Plane，
專案名稱：「2026 測試 OKR」
負責人：王大明
```

### 步驟 4: 體驗互動式拆分 ✨

當 Claude 遇到跨季度 Objective（如 Q1-Q4）時，會啟動互動流程：

```
🎯 發現跨季度 Objective：

Objective A：AI 輔助開發 (Q1-Q4)
時程: Q1-Q4（全年度）
Key Results:
  - KR1: 學習使用 Skills
  - KR2: 學習自架 MCP
  - KR3: 學習全自動開發流程

你希望如何拆分到各季度？

【選項 1】每季度重複相同的 Objective 和所有 KRs
【選項 2】將 KRs 分配到不同季度
【選項 3】在第一個季度創建，標註完整時程
【選項 4】自訂拆分

請選擇：1, 2, 3, 或 4
```

**推薦選擇：選項 2**（最靈活）

```
你選擇：2
```

接著 Claude 會逐個詢問：

```
📋 KR1: 學習使用 Skills

建議季度：Q1
  理由：基礎能力，適合早期建立
  
接受建議嗎？
  - 直接按 Enter 接受
  - 輸入季度編號調整 (例如: Q2)
  
你的選擇：[Enter]

✅ KR1 → Q1
```

最後確認：

```
✅ 分配完成：

Q1 Cycle:
  - [O] AI 輔助開發
    - [KR] 學習使用 Skills
    - [KR] 學習自架 MCP

Q3 Cycle:
  - [O] AI 輔助開發
    - [KR] 學習全自動開發流程

確認要這樣創建嗎？(yes/no)
```

輸入 `yes` 完成匯入！

## 📋 支援的格式

### ✅ 完整支援

**3 欄表格：**
```markdown
| KR  | 關鍵成果 | 衡量指標 |
| --- | ---- | ---- |
| KR1 | 完成培訓 | 產出 2 個模型 |
```

**2 欄表格：**
```markdown
| KR  | 關鍵成果 Key Result |
| --- | --------------- |
| KR1 | 完成培訓並產出 2 個模型   |
```

**組織級（4 欄）：**
```markdown
| KR  | 關鍵成果 | 時程 | 負責人 |
| --- | ---- | -- | --- |
| KR1 | 完成規劃 | Q1 | 小美  |
```

### ✅ 時程支援

**Objective 標題中：**
- `(Q1)`, `(Q2)`, `(Q3)`, `(Q4)` → 單季度
- `(Q1-Q2)` → 跨兩季度（觸發互動）
- `(Q1-Q4)`, `(1-12個月)` → 全年度（觸發互動）
- `(1-3個月)` = Q1
- `(4-6個月)` = Q2
- `(7-9個月)` = Q3
- `(10-12個月)` = Q4

## 🎨 互動式拆分選項詳解

### 選項 1：每季度重複
適用於：持續學習、橫向支援

結果：
```
Q1, Q2, Q3, Q4 都有完全相同的 Objective 和所有 KRs
```

### 選項 2：分配 KRs（推薦）✨
適用於：有階段性里程碑

結果：
```
Q1: [O] 目標 + KR1, KR2
Q3: [O] 目標 + KR3
```

### 選項 3：單季度創建
適用於：不想拆分，集中追蹤

結果：
```
Q1: [O] 目標 (全年度追蹤) + 所有 KRs
```

### 選項 4：自訂拆分
適用於：特殊需求

你描述，Claude 執行！

## 🆘 常見問題

**Q: 可以重新匯入嗎？**
A: Skill 會檢測重複，詢問你是使用現有還是創建新的。

**Q: 如果選錯拆分方式怎麼辦？**
A: 在 Plane 中手動調整，或刪除後重新匯入。

**Q: 負責人找不到怎麼辦？**
A: Skill 會列出可用成員讓你選擇，或暫時留空稍後手動設定。

**Q: 支援其他格式嗎？**
A: 目前主要支援表格格式。如果你的格式不同，告訴我，我可以調整！

## 🚀 進階功能

### 整合檔案匯入

如果你有整合多個來源的檔案（像 `2026.okr.platform.md`），skill 會：
1. 識別不同區塊
2. 適當處理組織級和個人級 OKR
3. 統一匯入到同一個專案

### 自訂 Work Item Types

如果你的 Plane 有自訂類型，可以在使用時指定：
```
使用 "Objective" 類型給 Objectives，
使用 "Key Result" 類型給 KRs
```
